## Development setup

```bash
# install dependencies
yarn

# serve with hot reload at localhost:4200
yarn dev

# lint
yarn lint

# build
yarn build

# local preview
yarn serve
```

## Actions setup

**Rule:**

- `any PR open` → `CI:Build test`
- `new tag v*` → `CI:Create Release`
- `release create` → `CI:Deploy` → `CI:Bundler` → `CI:Execute server script`
